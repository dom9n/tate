# Как выложить сайт на GitHub и чтобы фото подгружались

## 1. Создайте репозиторий на GitHub

- Зайдите на [github.com](https://github.com), создайте новый репозиторий (например, `tate-site`).
- Залейте туда все файлы проекта (index.html, menswear.html, styles.css, script.js, tate-photos.js, menswear-photo.js, папка uploads и т.д.).

## 2. Включите GitHub Pages

- В репозитории: **Settings** → слева **Pages**.
- В разделе **Source** выберите **Deploy from a branch**.
- В **Branch** выберите **main** (или **master**), папку **/ (root)**.
- Сохраните (**Save**).

Через 1–2 минуты сайт будет доступен по адресу:
`https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`

## 3. Добавьте фото — они подгрузятся сами

Сайт на GitHub Pages берёт картинки из папки **uploads** в этом же репозитории.

**Что нужно сделать:**

1. Откройте репозиторий на GitHub.
2. Зайдите в папку **uploads**.
3. Нажмите **Add file** → **Upload files**.
4. Загрузите файлы с **такими именами**:
   - **hero.png** — фото на главной странице;
   - **menswear_0.png** — первая плашка на Menswear;
   - **menswear_1.png**, **menswear_2.png**, **menswear_3.png**, **menswear_4.png** — остальные четыре плашки.
5. Нажмите **Commit changes**.

Через минуту обновите страницу сайта — фото появятся. С телефона и с любого устройства по ссылке `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/` будут видны те же фото.

**Важно:** имена файлов должны быть именно такими (латиницей, без пробелов). Формат — PNG.

## Если хотите заменить фото

Просто загрузите в папку **uploads** новый файл с тем же именем (например, hero.png) и сделайте коммит. На сайте подгрузится новое изображение.
